# UPI Transactions Report (Power BI)

This project visualizes digital payment activity using UPI (Unified Payments Interface) transaction data.  
The dashboard provides insights into transaction patterns, banking behavior, payment methods, customer demographics, and financial trends across multiple currencies.

This feature allows users to explore the dataset from different perspectives ‚Äî whether they prefer trend-based visuals (line charts) or comparative month-to-month values (column charts).

üí± Multi-Currency Filtering

The dataset includes transactions in four different currencies:
- USD
- EUR
- GBP
- INR

Users can select the desired currency from the Filter Pane to view metrics consistently in one unit.
This ensures that comparisons remain accurate and meaningful without currency-value confusion.

üéö Interactive Filters

Additional slicers are available to refine insights by:

- City
- Device Type
- Gender & Age Group
- Merchant Name
- Payment Method
- Purpose
- Transaction Type

These filters make the report highly adaptable for exploratory and targeted analysis.


## Page 2

This page displays a monthly summary of transaction amounts and remaining balances across key cities including Bangalore, Delhi, Hyderabad, and Mumbai. Each column represents a currency type (EUR, USD, GBP, INR), allowing users to easily compare financial activity across different locations. The table includes conditional color-highlighting, enabling users to quickly identify trends from the lowest to the highest values for clearer financial insights. Interactive slicers at the top allow filtering by parameters such as bank name, city, device type, gender, merchant name, payment method, and transaction type, ensuring a customized and detailed analytical view.
