# UPI Transactions Analysis Dashboard (Power BI)

This project visualizes digital payment activity using UPI (Unified Payments Interface) transaction data.  
The dashboard provides insights into transaction patterns, banking behavior, payment methods, customer demographics, and financial trends across multiple currencies.

üí± Multi-Currency Filtering

The dataset includes transactions in four different currencies:
- USD
- EUR
- GBP
- INR

Users can select the desired currency from the Filter Pane to view metrics consistently in one unit.
This ensures that comparisons remain accurate and meaningful without currency-value confusion.

üéö Interactive Filters

Additional slicers are available to refine insights by:

- City
- Device Type

- Merchant Name
- Payment Method
- Purpose
- Transaction Type

These filters make the report highly adaptable for exploratory and targeted analysis.
